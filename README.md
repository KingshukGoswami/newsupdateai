# newsupdateai
This ai will fetch the latest news globally and then send you a mail with 10 mails.

<p> user_config.json has the topics filtered by the users. 
The accepted values for the catagory are- <br>
[
  "business",
  "entertainment",
  "general",
  "health",
  "science",
  "sports",
  "technology"
]
</p>
